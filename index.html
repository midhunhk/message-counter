A Modern Android app for counting SMS sent in a cycle

Summary

Message Counter is a Free and Open Source Android App

It has been downloaded more than 95,000 times from the Google Play Store, translated into 5 languages and used in more than 10 countries.

Message Counter Version 4.0 is now available from the Google Play Store.

Features

  • SMS sent per each cycle
  • SMS sent each week
  • SMS sent in a year
  • Notification when SMS limit per cycle is reached
  • Top Message Senders
  • Material Design
  • Widgets
  • Multi language support (English, Spanish, French, Portuguese, German)
  • No Advertisements

Contribute

Message Counter is Open Source, with the complete source code hosted online at GitHub. You can contribute to this app by reporting new issues or improving the current code.

It is developed in native Android technology with the latest version of Android Studio and build tools.
