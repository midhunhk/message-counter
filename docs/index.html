Message Counter

Open Source app that counts your sent messages.

Message Counter is an Android app that counts the number of SMS sent in a cycle and notifies you when you have reached the limit. It is Open Source and has much more features.

Built with Android Studio

The latest version of Message Counter is build using Android Studio. The entire project and dependent libraries have been migrated to the gradle build system and built using the latest version of Andorid Studio and SDK tools.

Features

Message Counter provides the below features to its users:


  • Track the number of SMS messages in inbox, sent and total.
  • Track number of SMS sent per each cycle
  • Track number of SMS sent each week
  • Notification when SMS limit per cycle is reached
  • List of top contacts who have sent you messages
  • Material Design
  • Widgets
  • Multiple language support
  • No Advertisements
